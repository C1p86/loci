Import `YamlParseError`, `ConfigReadError`, `SecretsTrackedError` from '../errors.js'. Import type `ConfigLoader`, `ResolvedConfig`, `ConfigLayer` from '../types.js'.

2. **isEnoent(err: unknown): boolean** helper — checks `(err as NodeJS.ErrnoException).code === 'ENOENT'`.

3. **flattenToStrings(obj: Record<string, unknown>, filePath: string, prefix = ''): Record<string, string>** — Recursive dot-flatten per D-01. For each entry:
   - If value is string: set `result[fullKey] = value`
   - If value is a non-null, non-array object: recurse with `fullKey` as prefix
   - Else (number, boolean, array, null): throw `new YamlParseError(filePath, undefined, new Error(\`\${fullKey}: expected string, got \${Array.isArray(value) ? 'array' : typeof value}\`))` per D-04. For null specifically, use type string 'null'.
   - **Dot-collision detection:** Before setting `result[fullKey]`, check if `fullKey` already exists in `result`. If so, throw `new YamlParseError(filePath, undefined, new Error(\`Key collision: "\${fullKey}" appears both as nested path and direct key\`))`.

4. **readLayer(filePath: string | undefined, layer: ConfigLayer): { values: Record<string, string>; layer: ConfigLayer } | null** — Synchronous single-layer reader:
   - If `filePath` is undefined: return null
   - Try `readFileSync(filePath, 'utf8')`. On ENOENT: return null (D-08). On other error: throw `new ConfigReadError(filePath, err)`.
   - Call `parse(raw)` from yaml package. Wrap in try/catch: if `err instanceof YamlLibError`, extract `err.linePos?.[0]?.line` and throw `new YamlParseError(filePath, line, err)` (D-06, CFG-07).
   - If parsed is null/undefined: return `{ values: {}, layer }` (D-07 empty file).
   - If parsed is not a plain object or is an array: throw `new YamlParseError(filePath, undefined, new Error('Root document must be a YAML mapping'))`.
   - Call `flattenToStrings(parsed, filePath)` and return `{ values, layer }`.

5. **mergeLayers(layers: ReadonlyArray<...>): ResolvedConfig** — Iterate layers in order (machine, project, secrets, local). For each non-null layer, for each key/value: set `values[key] = value`, set `provenance[key] = entry.layer`. After iteration, build `secretKeys` as a Set of keys where `provenance[key] === 'secrets'` (final-provenance semantics per RESEARCH.md assumption A1). Return `{ values: Object.freeze(values), provenance: Object.freeze(provenance), secretKeys: Object.freeze(secretKeys) }`.

6. **isSecretTrackedByGit(cwd: string): boolean** — Synchronous check:
   - Try `execSync('git ls-files --error-unmatch .loci/secrets.yml', { stdio: 'pipe', cwd })`. If succeeds (exit 0): return true.
   - Catch: if `(err as {status?: number}).status === 128` OR `isEnoent(err)` (git not installed): return false. Otherwise return false (exit 1 = not tracked).

7. **configLoader export** — `export const configLoader: ConfigLoader = { async load(cwd) { ... } }`:
   - Compute paths: `machinePath = process.env['LOCI_MACHINE_CONFIG']` (undefined if not set), `projectPath = join(cwd, '.loci', 'config.yml')`, `secretsPath = join(cwd, '.loci', 'secrets.yml')`, `localPath = join(cwd, '.loci', 'local.yml')`.
   - Read all 4 layers: `const layers = [readLayer(machinePath, 'machine'), readLayer(projectPath, 'project'), readLayer(secretsPath, 'secrets'), readLayer(localPath, 'local')]`.
   - If `layers[2] !== null` (secrets layer loaded successfully): call `isSecretTrackedByGit(cwd)`. If returns true: `process.stderr.write('[loci] WARNING: .loci/secrets.yml is tracked by git. Run: git rm --cached .loci/secrets.yml\n')`. Do NOT throw (D-05, CFG-09).
   - Return `mergeLayers(layers)`.

**Key constraints:**
- Do NOT use `async readFile` — keep file reads synchronous since they're tiny config files and the load() function is already async at the interface level (future-proofing).
- Use `.js` suffix on all relative imports per verbatimModuleSyntax convention.
- Do NOT log secret values anywhere — error messages include key paths and types, never values.
- The `yaml` package's default `parse()` uses YAML 1.2 semantics, satisfying CFG-10 automatically.
