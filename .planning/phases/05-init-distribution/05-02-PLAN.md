From package.json:
- Install command: `npm i -g xci` (per D-01, package name is xci, binary is loci)

Config file paths (from src/config/index.ts):
- Machine: path from env var `LOCI_MACHINE_CONFIG`

Commands format (from src/commands/):
- Single: `cmd: ["program", "arg1"]` or `cmd: "program arg1"`
- Sequential: `steps: [alias1, alias2, "inline command"]`

- Platform overrides: `linux:`, `windows:`, `macos:` blocks
- Description: `description: "text"` shown in `loci --list`

CLI flags:
- `loci --list` or `loci -l`: show aliases
- `loci <alias> --dry-run`: preview resolved command
- `loci <alias> --verbose`: show config trace
- `loci <alias> -- --extra-args`: pass through to child
- `loci --version` or `loci -V`: show version
- `loci --help`: show help

Interpolation:
- `${VAR}` in commands resolves from merged config
- All config values injected as env vars to child processes
- Secrets redacted with *** in --dry-run and --verbose
