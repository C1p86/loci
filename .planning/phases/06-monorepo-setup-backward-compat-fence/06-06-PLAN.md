### Step 1.2 Run all 5 ROADMAP success criteria checks

Execute each of these checks in order and capture their output. STOP on the first failure and report which success criterion and plan is responsible.

```bash
# SC-1: v1 test suite passes in packages/xci/ layout (BC-02)
pnpm --filter xci test
SC1_EXIT=$?
# Expect: SC1_EXIT = 0

# SC-2a: dist/cli.mjs under 200 KB (BC-03 size)
BUNDLE_BYTES=$(wc -c < packages/xci/dist/cli.mjs)
echo "Bundle bytes: $BUNDLE_BYTES"
# Expect: BUNDLE_BYTES < 204800 and > 100000

# SC-2b: size-limit tool also confirms (D-15 parity with CI)
pnpm --filter xci size-limit
SC2_EXIT=$?
# Expect: SC2_EXIT = 0

# SC-3: ws and reconnecting-websocket NOT in dist/cli.mjs (BC-03 fence)
if grep -E "(reconnecting-websocket|['\"]ws['\"])" packages/xci/dist/cli.mjs; then
  echo "FAIL: fence leak"
  SC3_EXIT=1
else
  echo "OK: no ws/reconnecting-websocket in bundle"
  SC3_EXIT=0
fi
# Expect: SC3_EXIT = 0

# SC-4: turbo run build completes with all 3 packages
pnpm turbo run build 2>&1 | tee /tmp/turbo-build.log
SC4_EXIT=$?
grep -qE "xci.*build" /tmp/turbo-build.log && echo "OK: xci in build summary"
grep -qE "@xci/server.*build" /tmp/turbo-build.log && echo "OK: @xci/server in build summary"
grep -qE "@xci/web.*build" /tmp/turbo-build.log && echo "OK: @xci/web in build summary"
# Expect: SC4_EXIT = 0 AND all three greps succeed

# SC-5: hyperfine cold-start under 300ms on Linux (BC-04)
# If on Linux (or WSL2), run hyperfine locally. If on macOS/Windows, document that this gate is CI-only.
if command -v hyperfine >/dev/null 2>&1 && [ "$(uname -s)" = "Linux" ]; then
  hyperfine --runs 10 --warmup 3 --export-json /tmp/hyperfine-local.json 'node packages/xci/dist/cli.mjs --version'
  MEAN_MS=$(node -e "const r=require('/tmp/hyperfine-local.json'); console.log((r.results[0].mean*1000).toFixed(1))")
  echo "Local hyperfine mean: ${MEAN_MS}ms"
  # Expect: MEAN_MS < 300
else
  echo "SKIP: hyperfine not available or not on Linux. CI fence-gates job (D-17) is the authoritative gate. Documenting skip in VERIFICATION.md."
fi
```
