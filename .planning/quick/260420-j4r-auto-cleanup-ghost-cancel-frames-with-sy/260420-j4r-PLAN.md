| T-j4r-02 | Denial of Service | malicious server floods agent with cancel frames for unknown run_ids to cause agent to flood server with synthetic result frames (reflection amplification, 1:1 ratio) | accept | Amplification ratio is 1:1 (one cancel → one result).
